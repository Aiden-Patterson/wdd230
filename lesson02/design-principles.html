Rule of Thirds

BYU-I

Link

It's hard to show in a single screenshot, but the rule of thirds applies vertically in this case. The first section is the header, containing the menu and banner. The middle section is full of articles and links to relevant places. The bottom is the footer that has contact information and some more links. This division of thirds helps organize the content, and focus on what is in the center

PARC: Repetition

Ebay.com

Link

Ebay uses the same style and shapes for each group of objects. Each tab is the same rounded rectangle. Each banner is a rectangle wtith the same width. Each featured category is a circle with a two line description underneath.
